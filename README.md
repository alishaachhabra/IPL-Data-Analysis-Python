# IPL-Data-Analysis-Python
# 📊 IPL Data Analysis using Python

This project focuses on performing Exploratory Data Analysis (EDA) and Machine Learning on IPL ball-by-ball dataset.

## 🎯 Objective
To analyze team performance, scoring trends, and predict run categories using machine learning.

## 📂 Dataset
Ball-by-ball IPL dataset (2008–2025)

## 🔍 Key Analysis
- Team performance analysis
- Runs per over trends
- Dismissal type distribution
- Extras contribution
- Correlation heatmap

## 📈 Insights
- Strong scoring patterns across overs
- "Caught" is most common dismissal
- Clear performance differences between teams

## 🤖 Machine Learning
- Random Forest Classifier
- Run category prediction
- Handled class imbalance

## 🛠️ Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## 📌 Conclusion
This project demonstrates how data analysis and machine learning can be applied to sports analytics.
